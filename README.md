# hello-world

A small Python project that now includes a classic Snake game built with the standard `tkinter` GUI toolkit.

## Run the game

```bash
python3 snake.py
```

## Controls

- Arrow keys or `W`/`A`/`S`/`D`: move the snake
- `Space`: pause or resume
- `R`: restart
- `Esc`: quit

Eat the red food, avoid the walls, and do not run into your own tail. The snake gets a little faster each time it eats.
